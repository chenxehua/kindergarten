## 1. 测试范围与策略

| 测试类型 | 覆盖范围 | 占比 |

| UI测试 | 小程序/PC端界面交互 | 10% |

| 安全测试 | 鉴权、越权、数据安全 | 5% |

| 环境 | 用途 | 数据库 |
